# MaIA Frontend - React

Aplicação React que contém a página principal (Home) e o Header para o site da Agência MaIA.

## Estrutura Atual
O projeto foi migrado de HTML/Express para **React + Vite**.
Todas as seções desnecessárias foram removidas para mantermos apenas a hero section isolada (estática e com fundo em vídeo).

## Instalação e Execução

Para instalar as dependências:
```bash
npm install
```

Para rodar o servidor de desenvolvimento:
```bash
npm run dev
```

O projeto estará rodando localmente (normalmente em `http://localhost:5173`).
